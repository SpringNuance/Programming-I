MatchPanel

class MatchPanel extends Component

A MatchPanel is a GUI component that is capable of displaying the current status of a single Match. It uses one of two color schemes, depending on whether the displayed match has finished or not.

NOTE TO STUDENTS: In this course, you don't need to understand how this class works or can be used.
