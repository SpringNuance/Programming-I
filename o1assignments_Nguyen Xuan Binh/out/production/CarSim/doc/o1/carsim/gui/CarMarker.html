CarMarker

class CarMarker extends Marker

The class CarMarker represents the markers used in CarSim to represent cars graphically. These markers are linked with a destination marker and can move about, changing color as the car consumes fuel.

NOTE TO STUDENTS: In this course, you don't need to understand how this class works or can be used.
