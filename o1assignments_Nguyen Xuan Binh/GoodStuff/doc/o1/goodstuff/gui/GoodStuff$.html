GoodStuff

object GoodStuff extends SimpleSwingApplication with O1AppDefaults

The singleton object GoodStuff represents the GoodStuff experience diary application, and serves as an entry point for the program. It can be run to start up the GoodStuff user interface.

NOTE TO STUDENTS: In this course, you don't need to understand how this object works or can be used, apart from the fact that you can use this file to start the program.
